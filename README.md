# Suricata-IDS

Another exciting week of hands-on cybersecurity learning under the mentorship of **Abdullah Zia**.

This week, I:

* 🔹 Installed and configured **Suricata IDS**.
* 🔹 Wrote and tested custom detection rules for **Nmap SYN scans, HTTP directory traversal attacks, and ICMP flood attacks**.
* 🔹 Integrated **Suricata** with **Wazuh SIEM** and validated alert forwarding using **eve.json**.
* 🔹 Correlated network-based alerts with **File Integrity Monitoring (FIM)** events using **Wazuh Threat Hunting**.
* 🔹 Implemented **GeoIP-based traffic blocking** using **ipset** and **iptables**.
* 🔹 Explored **firewall rule management, domain blocking, and the practical limitations of GeoIP filtering**.

This week's work strengthened my practical knowledge of **Network Intrusion Detection, SIEM integration, threat detection, and firewall security**, bringing me one step closer to becoming a skilled **SOC Analyst**.

I look forward to continuing this learning journey and building stronger Blue Team skills. 💙
